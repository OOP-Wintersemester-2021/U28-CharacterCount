# U29-CharacterCount

In diesem Übungsblatt implementieren Sie eine Konsolenanwendung, die Eingaben von der Konsole liest und zählt, wie oft einzelne Buchstaben in der Eingabe vorkommen. Erstellen Sie dazu ein neues Projekt in IntelliJ.

* Verwenden Sie `java.util.Scanner`, um die Eingabe von der Konsole als `String` zu erhalten
* Verwenden Sie `java.util.HashMap`, um anhand des `String` von der Konsole zu zählen, wie oft jeder Buchstabe des `String` vorkommt.
* Geben Sie das Ergebnis der Zählung auf der Konsole aus.
